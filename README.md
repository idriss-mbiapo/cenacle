# Cenacle

Projet personnel : quand je débutais en programmation web.

## Description

Ce dépôt contient mes premiers pas en développement web. Il s’agit d’un ensemble de fichiers HTML, CSS et peut-être JavaScript que j’ai réalisés pour apprendre et m’exercer.

## Fonctionnalités

- Découverte des bases du HTML et du CSS
- Premiers essais de mise en page
- Expérimentations diverses en web

## Prérequis

- Un navigateur web moderne

## Installation

1. Clone le dépôt :
   ```bash
   git clone https://github.com/idriss902/cenacle.git
   ```
2. Ouvre le fichier `index.html` dans ton navigateur.
